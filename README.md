# Customer Retention & Churn Analysis

## Future Interns Data Science Internship

### Objective
Analyze customer churn patterns and identify key retention drivers.

### Dataset
Telco Customer Churn Dataset

### Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

### Key Insights
- Month-to-month customers show highest churn.
- Fiber optic customers have higher churn rates.
- Senior citizens are more likely to churn.
- New customers churn more frequently.
- Long-term contracts improve retention.

### Recommendations
- Promote annual subscriptions.
- Improve fiber service experience.
- Strengthen onboarding programs.
- Reward loyal customers.
- Offer personalized retention plans.

### Outcome
Developed a customer retention analysis report with actionable recommendations to reduce customer churn.
